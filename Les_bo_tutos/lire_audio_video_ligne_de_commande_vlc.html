Lire des fichiers audio ou vidéo en ligne de commande avec VLC

Title description, 05/01/2021

Prérequis:

  • Une distribution linux avec VLC installé.
  • C'est tout : )

Comme vous l'avez sans doutes déjà compris, on va utiliser VLC en ligne de commande afin de lire des fichiers vidéo ou audio qu'ils se trouvent sur votre machine ou qu'ils soient sur le web.

Voyons tout d'abord comment utiliser VLC en ligne de commande pour lire des fichiers.

cvlc Videos/Asteroid.mp4
VLC media player 3.0.11.1 Vetinari (revision )
[0000000001296f20] dummy interface: using the dummy interface module...

Ici on utilise cvlc, on a donc une fenêtre qui s'ouvre avec la vidéo mais aucune interface pour interagire avec elle, si nous avions un fichier audio, nous n'aurions même pas de fenêtre et seulement le son (les raccourcis habituels pour mettre en pause, avancer dans la video, afficher les sous titres, etc... devraient quand même marcher). Pour avoir l'interface de VLC il faut utiliser vlc au lieu de cvlc, toutefois nous continuerons ici avec cvlc. Notez que nvlc est un bon compromis car il ouvre une interface en ligne de commande ce qui peut être pratique.

Ici nous avions un fichier local, mais on peut egalement utiliser un URL:

cvlc https://acuissot.github.io/France_Imper/Shannon/Media/violinDrum.mp3
VLC media player 3.0.11.1 Vetinari (revision )
[0000000001296f20] dummy interface: using the dummy interface module...

et nous pouvons même utiliser des URL youtube par exemple, on notera que beaucoup de vidéo (surtout musicales) ne sont pas accessible via cette méthode.

cvlc https://www.youtube.com/watch?v=kID5ugIuHig
VLC media player 3.0.11.1 Vetinari (revision )
[0000000001296f20] dummy interface: using the dummy interface module...

C'est bien gentil tout ça mais on va regarder un peu les options que propose VLC afin d'avoir un lecteur sur mesure et de pouvoir utiliser des playlists. Nous ne verrons pas toutes les options mais sachez qu'il y en a pour afficher des sous titres, changer de langue (si vous avez plusieurs pistes audio) et beaucoup d'autres, n'hésitez pas a utiliser vlc -hpour la page d'aide, vlc -H pour avoir la liste exhaustive de ces options et modules, vlc -l pour la liste des modules disponible (ou vlc --list-verbose pour avoir également une description de ces module) vlc -p moduleName pour une aide sur un module en particulier. Les modules peuvent servir à différentes choses comme ajouter des filtres audio ou video, utiliser des fonctions d'encodage ou de décodage particulière, importer une playlist iTunes et bien d'autres.

L'option --no-audio permet bien évidément de ne pas lancer la partie audio du fichier.

L'option -f lance la fenêtre en mode plein écran

Les options -Z, -L et -R permettent respectivement d'activer la lecture aléatoire, la lecture en boucle de la liste et la lecture en boucle du fichier actuellement lu.

Nous allons ici utiliser l'option --play-and-exit pour fermer l'application une fois la lecture terminée mais libre à vous de ne pas l'utiliser.

Pour lire plusieurs fichier, vous pouvez simplement utiliser:

cvlc media1 media2
VLC media player 3.0.11.1 Vetinari (revision )
[0000000001296f20] dummy interface: using the dummy interface module...

Cependant, il est plus propre de le faire ainsi:

cvlc --started-from-file --playlist-enqueue $(find . -type f | sort)
VLC media player 3.0.11.1 Vetinari (revision )
[0000000001296f20] dummy interface: using the dummy interface module...

où en l'occurence nous utilisons la commande find pour récupere les fichiers du répertoire et nous les trions avant de les lire en tant que playlist.
