Visualiseur audio en Vulkan


Le projet est la création d'un visualiseur audio de style vu-mètre ou peak-metre en utilisant l'API Vulkan. La difficulté réside dans l'utilisation d'une API qui ne m'était pas familière mais nous étions 3 camarades pour essayer de comprendre le fonctionnement. A l'origine, le son aurait dû être synchronisé avec le visuel à l'aide de VST mais cette partie est encore manquante aujourd'hui. Le code n'est pas présent sur git car le projet à été réalisé pour une entreprise qui ne souhaitais pas le rendre public. Le gogniomètre à été réalisé par un de mes camarades.


1 min

Explorageon


Explorageon est un jeu d'exploration développé à 5 étudiants en OpenGL, le but est de retrouver l'anneau sur la carte 3D (uniquement une carte de reliefs pur le moment) en la survolant. Le jeu est peu développé mais l'interêt réside dans le fait que la carte est générérée aléatoirement à chaque partie. Le code n'est pas présent sur git pour le moment.

Pow'Earth


Pow'Earth est une application web réalisée par notre équipe d'étudiants de l'Ensicaen et de l'Université Normandie au dôme avec l'aide des DUTs info-comm. Elle permet de faire des simulations du point de vu d'un magnat de l'énergie pour voir les impacts de ses choix sur ses concurrents, les populations locales, les gouvernements, sa production, les reserves, les lobbys, ... Le choix de se developper sur une seule ressource, de s'étendre géographiquement et agressivement et d'exporter les ressources est il la meilleure stratègie ? Difficile à savoir sans tester les differentes possibilités.

Echo Sciences Normandie, l'article sur ce projet et d'autres projets réalisés cette même semaine
